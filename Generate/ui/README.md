# v2/ui — PyQt5 UI for GMP Generator Engine

This is a minimal PyQt5 UI project scaffold.

Layout:
- Top: horizontal toolbar
- Middle: central area split horizontally; left = 1/3, right = 2/3
- Bottom: horizontal information/status bar

Run:

Windows / general:
```powershell
pip install -r v2/ui/requirements.txt
python v2/ui/main.py
```

Notes:
- The main window is implemented in `ui_main.py`.
- Adjust or extend widgets as needed.
