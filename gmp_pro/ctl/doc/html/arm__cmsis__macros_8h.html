Detailed Description

Defines a set of macros for floating-point arithmetic using the ARM CMSIS-DSP library.

This file provides the concrete implementation of the control library's math functions using the ARM CMSIS-DSP library for Cortex-M processors. It maps the generic control math macros to their corresponding hardware-accelerated CMSIS-DSP functions. This implementation is based on float32_t.
