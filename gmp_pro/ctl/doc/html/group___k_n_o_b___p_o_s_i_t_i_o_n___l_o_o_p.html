Detailed Description

A position controller that simulates discrete detents.

This controller calculates a target position corresponding to the nearest "snap" point and uses a P-controller to generate a restoring force (as a speed command) towards it.

This module creates a "knob" effect by defining a series of stable setpoints (detents). When the motor's actual position is near a detent, a proportional controller generates a speed reference to pull the motor towards the center of that detent, simulating the feel of a physical knob.
