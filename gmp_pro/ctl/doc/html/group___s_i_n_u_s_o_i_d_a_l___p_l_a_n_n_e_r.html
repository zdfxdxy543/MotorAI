Detailed Description

Generates cycloidal profiles for extremely smooth motion.

This module generates a position profile based on a sinusoidal acceleration profile (specifically, a cycloidal profile). This ensures that velocity and acceleration are zero at both the start and end of the motion, providing extremely smooth movement with continuous jerk. The planner generates a trajectory to move from a starting point to a target point over a specified duration.

The trajectory is based on a cycloidal profile, where the position \(p(t)\) is given by:

\[ p(t) = p_{start} + \Delta p \left( \frac{t}{T} - \frac{1}{2\pi} \sin\left(\frac{2\pi t}{T}\right) \right) \]

This results in a sinusoidal velocity profile:

\[ v(t) = \frac{\Delta p}{T} \left( 1 - \cos\left(\frac{2\pi t}{T}\right) \right) \]

And a sinusoidal acceleration profile:

\[ a(t) = \frac{2\pi \Delta p}{T^2} \sin\left(\frac{2\pi t}{T}\right) \]
