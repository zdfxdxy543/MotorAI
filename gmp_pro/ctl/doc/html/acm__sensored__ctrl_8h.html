Detailed Description

Implements a complete sensored Field-Oriented Control (FOC) for AC Induction Motors (ACM).

Author
Javnson (javns.nosp@m.on@z.nosp@m.ju.ed.nosp@m.u.cn)

This file provides a comprehensive controller for AC induction motors, including current and speed control loops, coordinate transformations, and flux estimation. It is designed to be highly configurable through preprocessor macros.
