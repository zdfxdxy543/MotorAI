Converts current controller PI gains from SI units to per-unit for an ACM.

The current controller's input is current error and output is voltage. Formulas:

\[ K_{p,pu} = \frac{K_{p,real}}{Z_{base}} \]

\[ K_{i,pu} = \frac{K_{i,real}}{Z_{base} \cdot \omega_{base}} \]

Parameters
[in]puPointer to the populated per-unit consultant structure.
[in]kp_realThe real-world proportional gain (V/A).
[in]ki_realThe real-world integral gain (V/(As)).
[out]kp_puPointer to store the per-unit proportional gain.
[out]ki_puPointer to store the per-unit integral gain.

Converts speed controller PI gains from SI units to per-unit for an ACM.

The speed controller's input is speed error and output is current. Formulas:

\[ K_{p,pu} = \frac{K_{p,real} \cdot \omega_{base}}{I_{base}} \]

\[ K_{i,pu} = \frac{K_{i,real}}{I_{base}} \]

Parameters
[in]puPointer to the populated per-unit consultant structure.
[in]kp_realThe real-world proportional gain (A/(rad/s)).
[in]ki_realThe real-world integral gain (A/rad).
[out]kp_puPointer to store the per-unit proportional gain.
[out]ki_puPointer to store the per-unit integral gain.
