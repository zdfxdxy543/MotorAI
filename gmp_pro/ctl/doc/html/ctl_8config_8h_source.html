13// ctl_ : as prefix
14// <action>_ : such as step(calculate one step), get, init, setup, and so on.
15// <target object> : such as ramp, sincos, track_pid, and so on.
16// _via_<data_source> : [optional] to specify the data source
17// such as angle, pos_encoder, and so on.
18// _<configurations> : [optional] to specify the key point of this function.
