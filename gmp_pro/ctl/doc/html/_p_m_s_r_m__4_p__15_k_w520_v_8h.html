Detailed Description

Defines parameters for a 15kW Permanent Magnet Synchronous Reluctance Motor (PMSRM).

This file contains the electrical, mechanical, and operational parameters for a specific high-power, salient-pole PMSM. These macros are intended to be used throughout the motor control application to configure various algorithms (like MTPA) and safety limits.
