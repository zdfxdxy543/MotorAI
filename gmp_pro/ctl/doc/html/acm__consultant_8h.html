GMP_STATIC_INLINE parameter_gt ctl_consult_acm_Damp_pu ( ctl_per_unit_consultant_t * pu,
im_dsn_consultant_t * acm )

Converts the damping factor (B) [Nms] to its per-unit equivalent.

The per-unit damping is derived from the friction torque equation T_damp = B * ω_m. In per-unit, T_damp_pu = B_pu * ω_m_pu. This leads to the base damping B_base = T_base / omega_m_base.

GMP_STATIC_INLINE parameter_gt ctl_consult_acm_Inertia_pu ( ctl_per_unit_consultant_t * pu,
im_dsn_consultant_t * acm )

Converts the moment of inertia (J) [kg m^2] to its per-unit equivalent.

The per-unit inertia is derived from the mechanical equation T = J * domega/dt. In per-unit, T_pu = J_pu * d(omega_pu)/d(t_pu). Base time is defined as t_base = 1 / omega_e_base. This leads to the base inertia J_base = T_base / (omega_m_base * omega_e_base).
