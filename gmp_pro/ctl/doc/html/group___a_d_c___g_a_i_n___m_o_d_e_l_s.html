Detailed Description

A collection of functions to determine the correct gain for sensor signal conditioning.

The gain is a crucial parameter for converting a per-unit ADC value back into a physical quantity (e.g., Volts, Amperes). These models simplify the process of determining the correct gain based on hardware components. These models calculate the gain required by the ADC channel modules to scale the normalized ADC output into a meaningful physical unit.

parameter_gt Base_physical )

A generic function to calculate the required system gain.

This function calculates the gain needed to convert a per-unit ADC value to a final value scaled by a physical base. The formula is:

\[\text{gain} = \frac{V_{\text{ref}}}{G_{\text{sensor}} \cdot \text{Base}_{\text{physical}}} \]

Parameters
VrefThe ADC reference voltage (e.g., 3.3V).
G_sensorThe hardware gain of the analog front-end (e.g., the ratio of a voltage divider or the V/A sensitivity of a current sensor).
Base_physicalThe maximum physical value that corresponds to 1.0 p.u. in the control system (e.g., 400V or 50A).
Returns
The calculated gain for the ADC channel module.

Calculates gain for a current sensor using a shunt resistor and an operational amplifier.

The sensor measures current via the voltage drop across a shunt resistor, which is then amplified by an op-amp. The hardware gain is \( G_{\text{sensor}} = R_{\text{shunt}} \cdot G_{\text{opamp}} \) (in V/A).

Parameters
VrefThe ADC reference voltage.
IbaseThe maximum current to be measured (the physical base).
R_shuntThe resistance of the shunt resistor in Ohms.
G_opampThe voltage gain of the operational amplifier.
Returns
The calculated gain for the ADC channel module.

GMP_STATIC_INLINE parameter_gt ctl_gain_calc_volt_divider ( parameter_gt Vref,
parameter_gt Vbase,
parameter_gt R1,
parameter_gt R2 )

Calculates gain for a voltage sensor using a resistive divider.

The sensor hardware consists of two resistors, R1 and R2, forming a voltage divider. The hardware gain of this stage is \( G_{\text{sensor}} = \frac{R_2}{R_1 + R_2} \).

Parameters
VrefThe ADC reference voltage.
VbaseThe maximum voltage to be measured (the physical base).
R1The resistance of the resistor connected between the source voltage and the ADC input.
R2The resistance of the resistor connected between the ADC input and ground.
Returns
The calculated gain for the ADC channel module.
