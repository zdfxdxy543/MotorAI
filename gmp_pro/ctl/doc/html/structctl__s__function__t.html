Detailed Description

Data structure for a 2-Pole-2-Zero S-domain transfer function.

This structure holds the discretized Z-domain coefficients and the state variables needed to implement the difference equation. The underlying continuous-time transfer function is of the form:

\[ H(s) = K \frac{(s/\omega_{z1}+1)(s/\omega_{z2}+1)}{(s/\omega_{p1}+1)(s/\omega_{p2}+1)} \]

This is converted to a Z-domain transfer function:

\[ H(z) = \frac{b_0 + b_1z^{-1} + b_2z^{-2}}{1 + a_1z^{-1} + a_2z^{-2}} \]
