Detailed Description

Defines a 3x3 matrix type and related mathematical operations.

This file provides a standard implementation for 3x3 matrix arithmetic, which is fundamental for linear transformations in 3D space, such as rotations. The matrix elements are stored in row-major order: [m00, m01, m02, m10, m11, m12, m20, m21, m22].
