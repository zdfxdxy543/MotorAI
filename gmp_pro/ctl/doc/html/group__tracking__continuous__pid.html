Detailed Description

A composite PID controller for smooth setpoint tracking using a continuous-form PID.

This file implements a tracking PID controller based on the continuous-form PID implementation. It combines three intrinsic modules:

  1. A Frequency Divider: To execute the controller at a lower rate.
  2. A Slope Limiter: To generate a smooth trajectory (ramp) for the setpoint.
  3. A Continuous-Form PID: To calculate the control output. This is ideal for setpoint tracking where smooth transitions are needed.

ctrl_gt target,
ctrl_gt fbk )

Executes one step of the tracking continuous PID controller.

This function should be called at the main system frequency (fs). It uses an internal divider to decide when to execute the core control logic. When the divider triggers, it updates the setpoint trajectory and calculates a new PID output. On other steps, it holds the last calculated PID output.

[in]targetThe ultimate target value for the setpoint.
