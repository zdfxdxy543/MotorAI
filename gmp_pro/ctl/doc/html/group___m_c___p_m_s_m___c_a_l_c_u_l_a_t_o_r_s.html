Calculates the PMSM back-EMF constant (Ke) in V_RMS / kRPM.

This is a common unit found on motor datasheets. Formula:

\[ K_e [V_{RMS,L-L}/kRPM] = p \cdot \psi_f \cdot \frac{\pi \sqrt{3}}{3} \]

Estimates PI controller parameters for the d-q axis current loops.

Uses the bandwidth method (pole placement) for tuning. The plant is modeled as a first-order system. Formulas:

\[ K_p = \alpha_c L, \quad K_i = \alpha_c R_s \]

where \(\alpha_c\) is the desired bandwidth.

Parameters
[in]pmsm_dsnPointer to the populated motor design consultant structure.
[in]bandwidth_hzThe desired closed-loop bandwidth in Hertz (Hz).
[out]kp_dPointer to store the d-axis proportional gain.
[out]ki_dPointer to store the d-axis integral gain.
[out]kp_qPointer to store the q-axis proportional gain.
[out]ki_qPointer to store the q-axis integral gain.

Estimates PI controller parameters for the speed loop.

Uses the bandwidth method. Assumes the current loop is significantly faster. Formulas:

\[ K_p = \frac{\alpha_s J}{K_t}, \quad K_i = \frac{\alpha_s B}{K_t} \]

where \(\alpha_s\) is the desired bandwidth.

Parameters
[in]pmsm_dsnPointer to the populated motor design consultant structure.
[in]bandwidth_hzThe desired closed-loop bandwidth in Hertz (Hz).
[out]kpPointer to store the calculated proportional gain.
[out]kiPointer to store the calculated integral gain.
