GMP_STATIC_INLINE void gmp_base_ctl_step ( void )

The main kernel step function for the GMP CTL module.

This function is the primary entry point intended to be invoked by the user in the main control ISR. It conditionally compiles the appropriate dispatch logic based on the selected framework.
