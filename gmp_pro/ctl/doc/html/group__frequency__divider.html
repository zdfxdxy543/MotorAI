Detailed Description

A simple counter-based module to divide a clock or event frequency.

This module implements a basic frequency divider (or prescaler). It is used to generate a periodic trigger or event at a frequency that is an integer fraction of a higher input frequency. For example, it can be used to execute a slow control loop every N calls of a faster main ISR.

Executes one step of the frequency divider.

Increments the internal counter. When the counter reaches the target value, it resets the counter and returns 1. Otherwise, it returns 0.
