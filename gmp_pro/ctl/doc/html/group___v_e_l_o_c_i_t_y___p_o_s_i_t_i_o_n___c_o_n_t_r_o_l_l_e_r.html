Detailed Description

A simple velocity and position loop.

This module provides a simple and effective position controller. It takes a target position (composed of full revolutions and an angle) and a feedback position, calculates the error, and multiplies it by a proportional gain (Kp) to produce a speed reference. The output is saturated to a defined limit. This controller forms the outermost loop in a cascaded position control system.
