Detailed Description

Generates PWM signals for a single-phase H-bridge inverter.

This module generates two PWM compare values for a standard single-phase H-bridge inverter using unipolar Sine Pulse Width Modulation (SPWM). It accepts a modulation signal from -1.0 to 1.0.

It also includes a dead-time compensation feature that adjusts the PWM duty cycles based on the direction of the output current. This minimizes voltage distortion caused by the blanking time inserted by the PWM hardware.
