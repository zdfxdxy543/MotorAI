Detailed Description

A library of discrete IIR filters for control loop compensation.

This file contains implementations for discrete-time lead and lag compensators. These are first-order IIR filters used to improve the stability (phase margin) and steady-state performance of a control loop. The continuous-time transfer functions are discretized using the Bilinear Transform:

\[ s = \frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}} \]

The continuous-time transfer function is:

\[ H(s) = \frac{(\tau_D + K_D)s + 1}{\tau_D s + 1} \]

After discretization, it becomes a 1-Pole-1-Zero (1P1Z) filter.
