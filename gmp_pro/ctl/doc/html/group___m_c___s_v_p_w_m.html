Detailed Description

This module contains the structures and functions for SVPWM calculations.

The SVPWM module converts a two-phase orthogonal stationary reference frame vector \((U_\alpha, U_\beta)\) into three-phase PWM signals. The process is typically divided into two stages:

  1. Calculation Stage: Transforms the \((U_\alpha, U_\beta)\) vector into intermediate duty cycle timings (Ta, Tb, Tc).
  2. Modulation Stage: Converts the intermediate timings into final PWM compare values for the timer peripheral.

GMP_STATIC_INLINE void ctl_svpwm_calc ( ctl_svpwm_channel_t * svpwm)

(Stage I) Calculates three-phase duty ratios from the alpha-beta vector.

This function implements an SVPWM algorithm by calculating an equivalent three-phase system and injecting a common-mode voltage to center the signals. This method is equivalent to centering the pulse widths in the PWM period.

The transformation from ($$U_\alpha$$, $$U_\beta$$) to the intermediate phase voltages ($$U_a$$, $$U_b$$, $$U_c$$) is as follows:

\[U_a = U_\alpha, \\ U_b = -U_\alpha /2 + \sqrt{3}/2\cdot U_\beta, \\ U_c = -U_\alpha /2 - \sqrt{3}/2\cdot U_\beta, \]

Parameters
[in,out]svpwmPointer to the SVPWM channel structure. The inputs ctl_svpwm_channel_t::Ualpha and ctl_svpwm_channel_t::Ubeta are used, and the result is stored in the ctl_svpwm_channel_t::T array.

GMP_STATIC_INLINE void ctl_svpwm_calc2 ( ctl_svpwm_channel_t * svpwm)

(Stage I) Calculates three-phase duty ratios using a sector-based algorithm.

This function provides an alternative implementation for the Stage I calculation. It determines the sector based on the reference vector and calculates the dwell times for the adjacent space vectors (T1, T2). These timings are then used to generate the final duty cycle ratios for each phase.

Parameters
[in,out]svpwmPointer to the SVPWM channel structure. The inputs ctl_svpwm_channel_t::Ualpha and ctl_svpwm_channel_t::Ubeta are used, and the result is stored in the ctl_svpwm_channel_t::T array.

GMP_STATIC_INLINE void ctl_svpwm_modulation ( ctl_svpwm_channel_t * svpwm)

(Stage II) Generates final PWM compare values from duty cycle timings.

This function converts the intermediate duty cycle timings (ctl_svpwm_channel_t::T array, nominally in the range [-0.5, 0.5]) into absolute PWM compare values. It does this by shifting the range to [0, 1.0] and then scaling by the PWM period.

Parameters
[in,out]svpwmPointer to the SVPWM channel structure. The ctl_svpwm_channel_t::T array and ctl_svpwm_channel_t::pwm_period are used, and the result is stored in ctl_svpwm_channel_t::pwm_cmp.
