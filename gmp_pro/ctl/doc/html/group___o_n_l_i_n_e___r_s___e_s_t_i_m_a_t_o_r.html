Detailed Description

An online estimator for stator resistance based on the Model Reference Adaptive System principle.

This module continuously updates the estimated stator resistance (Rs) to track changes due to temperature variations during motor operation. It uses the d-axis voltage equation as the reference model and a PI controller as the adaptation mechanism.
