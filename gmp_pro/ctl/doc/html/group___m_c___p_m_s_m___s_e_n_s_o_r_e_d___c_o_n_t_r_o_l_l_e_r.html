Detailed Description

A complete Field-Oriented Controller for Permanent Magnet Synchronous Motors.

PMSM Bare-metal Controller Usage Guide:

  1. Attach Physical Interfaces:
  • Use ctl_attach_pmsm_bare_output to attach the PWM output interface.
  • Use functions within the mtr_interface to attach input interfaces like ADC, encoders, etc.
  1. Initialize the Controller:
  • Fill the pmsm_bare_controller_init_t struct to specify motor and controller parameters.
  • Call ctl_init_pmsm_bare_controller to initialize the controller entity.
  1. Select Operating Mode and Provide a Target: This controller supports various operating modes. Enter a mode by calling its corresponding switching function and provide a target value using a set function.
  • V”┴”┬ Mode: ctl_pmsm_ctrl_valphabeta_mode, ctl_set_pmsm_ctrl_valphabeta
  • Voltage (Vdq) Mode: ctl_pmsm_ctrl_voltage_mode, ctl_set_pmsm_ctrl_vdq_ff
  • Current (Idq) Mode: ctl_pmsm_ctrl_current_mode, ctl_set_pmsm_ctrl_idq_ff
  • Velocity Mode: ctl_pmsm_ctrl_velocity_mode, ctl_set_pmsm_ctrl_speed
  • Position Mode: ctl_pmsm_ctrl_position_mode, set_pmsm_ctrl_position
Note
The mode switching functions only change internal flags. For smooth transitions during runtime, an additional transition algorithm should be implemented.
  1. Invoke in the Main Interrupt Service Routine (ISR):
  • First, call the motor interface's step function to update inputs (e.g., ADC sampling).
  • Then, call ctl_step_pmsm_ctrl to execute the core FOC calculations.
  • Finally, call the PWM interface's function to output the modulation result.
  1. Enable/Disable the Controller:
  • Use ctl_enable_pmsm_ctrl to enable the controller.
  • Use ctl_disable_pmsm_ctrl to disable the controller.
  • It is recommended to call ctl_clear_pmsm_ctrl to clear internal states before enabling.
