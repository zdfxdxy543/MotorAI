Detailed Description

A library of modules for generating standard test waveforms.

This file contains implementations for common signal generators used for simulation, testing, and control system stimulation. It includes a sine/cosine wave generator based on an iterative phasor rotation and a ramp wave generator.

GMP_STATIC_INLINE void ctl_step_sine_generator ( ctl_sine_generator_t * sg)

Executes one step of the sine/cosine generator.

Updates the sine and cosine outputs by rotating the phasor by the step angle. This is based on the angle sum identities: sin(A+B) = sin(A)cos(B) + cos(A)sin(B) cos(A+B) = cos(A)cos(B) - sin(A)sin(B)

Parameters
[in,out]sgPointer to the sine generator instance.
