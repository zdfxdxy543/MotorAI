Detailed Description

Data structure for the DF22 controller.

This structure implements a second-order IIR filter using the Direct Form II topology. This form is canonical, meaning it uses the minimum number of delay elements for a given transfer function.

The transfer function is:

\[ H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2}}{1 + a_1 z^{-1} + a_2 z^{-2}} \]
