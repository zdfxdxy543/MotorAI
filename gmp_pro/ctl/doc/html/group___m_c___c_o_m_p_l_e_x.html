GMP_STATIC_INLINE ctl_complex_t ctl_cdiv ( ctl_complex_t a,
ctl_complex_t b )

Divides one complex number by another.

The division is performed as:

\[\frac{a}{b} = \frac{a \cdot b^*}{|b|^2} \]

Parameters
aThe numerator.
bThe denominator.
Returns
The result of the division (a / b). Returns {0, 0} if the denominator is zero.

GMP_STATIC_INLINE ctl_complex_t ctl_cmul ( ctl_complex_t a,
ctl_complex_t b )

Multiplies two complex numbers.

The multiplication is performed as:

\[(a_r + j a_i)(b_r + j b_i) = (a_r b_r - a_i b_i) + j(a_r b_i + a_i b_r) \]

Parameters
aThe first complex number.
bThe second complex number.
Returns
The result of the multiplication (a * b).
