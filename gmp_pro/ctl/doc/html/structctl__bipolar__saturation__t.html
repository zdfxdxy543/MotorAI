Detailed Description

Data structure for a bipolar saturation block.

This block applies different saturation bands for positive and negative inputs. For positive inputs, the output is clamped to [out_min, out_max]. For negative inputs, the output is clamped to [-out_max, -out_min]. This is useful for creating a dead-zone around zero if out_min > 0.
