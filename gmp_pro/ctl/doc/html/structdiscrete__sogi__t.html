Detailed Description

SOGI Continuous-Time Transfer Functions.

A SOGI provides two outputs from a single input R(s). The direct output D(s) is in-phase with the input's fundamental component and acts as a band-pass filter. The quadrature output Q(s) lags the direct output by 90 degrees and acts as a low-pass filter.

Band-Pass Filter Transfer Function (Direct Output D(s)):

\[ \frac{D(s)}{R(s)} = \frac{k \omega_0 s}{s^2 + k \omega_0 s + \omega_0^2} \]

Low-Pass Filter Transfer Function (Quadrature Output Q(s)):

\[ \frac{Q(s)}{R(s)} = \frac{k \omega_0^2}{s^2 + k \omega_0 s + \omega_0^2} \]

where \( \omega_0 \) is the center frequency and \( k \) is the damping coefficient.
