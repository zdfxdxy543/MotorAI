Detailed Description

Data structure for the generic FIR filter.

This structure holds the configuration and state for a direct-form FIR filter. The difference equation is:

\[ y(n) = \sum_{k=0}^{N-1} b_k x(n-k) \]

where N is the number of taps (order), b_k are the filter coefficients, and x are the input samples.
