Detailed Description

Data structure for the three-phase bridge SPWM modulation module.

Note
Recommended Value for Current Dead band:
  • Theoretical Minimum: I_min = 2 * Coss * Vdc / T_dt
  • Practical Setting: Typically 1% to 3% of the rated current.
  • Tuning Guide: Set higher if zero-crossing oscillation occurs; set lower if crossover distortion is observed. Ensure value is strictly larger than the ADC noise floor.
