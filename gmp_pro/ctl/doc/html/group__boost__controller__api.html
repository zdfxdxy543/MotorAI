Detailed Description

Functions and data structures for Boost converter control.

Contains all configurations, data structures, and functions for the Boost controller.

Boost Controller Usage Guide

This module implements a standard dual-loop (voltage and current) controller for a DC-DC Boost converter.

1. Initialization

  • Call ctl_init_boost_ctrl to configure the PID parameters for both the outer voltage loop and the inner current loop.
  • Call ctl_attach_boost_ctrl_input to link the controller to the ADC interfaces for sensing input/output voltages and inductor current.

2. Mode Selection

The controller supports several operating modes:

  • Voltage Mode: Closed-loop voltage control. Use ctl_boost_ctrl_voltage_mode to enter and ctl_set_boost_ctrl_voltage to set the target output voltage.
  • Current Mode: Closed-loop current control (voltage loop is bypassed). Use ctl_boost_ctrl_current_mode to enter and ctl_set_boost_ctrl_current to set the target inductor current.
  • Open Loop Mode: Directly control the output voltage command. Use ctl_boost_ctrl_openloop_mode to enter and ctl_set_boost_ctrl_voltage_openloop to set the voltage command.

3. Execution

  • Call ctl_step_boost_ctrl periodically in a timer ISR to execute one step of the control logic.
  • Call ctl_get_boost_ctrl_modulation to retrieve the calculated PWM duty cycle and apply it to your PWM peripheral.

4. Control

  • Use ctl_enable_boost_ctrl and ctl_disable_boost_ctrl for master control.
  • Use ctl_clear_boost_ctrl to reset PID integrators before enabling.

◆ CTL_BOOST_CTRL_POSITION

#define CTL_BOOST_CTRL_POSITION   UPPER_BRIDGE

Defines which switch in the half-bridge is being controlled by the PWM signal.

  • UPPER_BRIDGE: The duty cycle D controls the upper switch. Transfer function M = 1/D.
  • LOWER_BRIDGE: The duty cycle D controls the lower switch. Transfer function M = 1/(1-D).
