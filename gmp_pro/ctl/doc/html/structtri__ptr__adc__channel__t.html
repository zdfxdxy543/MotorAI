Detailed Description

Data structure for a three-channel pointer-based ADC module.

The raw data is accessed from a continuous array via a pointer. \(\text{value}[i] = \text{gain}[i] \cdot (\text{raw}_{\text{scaled}}[i] - \text{bias}[i]) \)
