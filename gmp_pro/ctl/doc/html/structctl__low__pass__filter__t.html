Detailed Description

A first-order infinite impulse response (IIR) low-pass filter.

This filter is implemented using the following difference equation, which is derived from the Z-transform of the continuous-time transfer function.

Continuous-time transfer function:

\[ H(s) = \frac{\omega_c}{s+\omega_c} \]

After discretization using a Zero-Order Hold (ZOH) equivalent, the Z-domain transfer function is:

\[ H(z) = \frac{1-e^{-\omega_c T_s}}{1-e^{-\omega_c T_s} z^{-1}} \]

This leads to the difference equation:

\[ y(n) = a \cdot x(n) + (1-a) \cdot y(n-1) \]

where \( a = 1 - e^{-\omega_c T_s} \). For small \( \omega_c T_s \), this can be approximated as \( a \approx \omega_c T_s \).
