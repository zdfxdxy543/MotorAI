Detailed Description

A module for detecting voltage sag and swell power quality events.

This module monitors a sinusoidal input voltage to detect power quality events such as sags (undervoltage) and swells (overvoltage). It uses a Second-Order Generalized Integrator (SOGI) to extract the fundamental amplitude of the signal. A fault is flagged if the amplitude remains outside the configured thresholds for a specified duration.
