1%% @brief Generate a Fuzzy Logic Look-Up Table (LUT) from a .fis file.
2%
3% @details This function aims to improve the runtime efficiency of a fuzzy
4% controller. It pre-calculates the outputs of a fuzzy system for all
5% discrete input points, generating a look-up table (control surface).
6% In a real-time control loop, this allows for rapid calculation of the
7% control output via linear interpolation, avoiding complex real-time
8% fuzzy inference.
9%
10% @param fis_file (string) The path and name of the .fis file.
11% @param dim_input (integer) The number of input variables (dimensions).
12% @param range_input (matrix) An Nx2 matrix specifying the value range for
13% each input variable. Each row corresponds to an
14% input, e.g., `[-1, 1]`.
15% @param mash_input (vector) A vector with N elements, specifying the number
16% of segments for each input range.
17%
18% @retval fuzzy_matrix The generated Look-Up Table (LUT), a 2D matrix.
19% @retval dim1_mesh The meshgrid vector for the first input dimension.
20% @retval dim2_mesh The meshgrid vector for the second input dimension.
21%
22% @note This version only supports fuzzy systems with 2 inputs and 1 output.
23%
24% @example
25% % Define input range as [-1, 1] and divide each dimension into 50 segments.
26% range = [-1, 1; -1, 1];
27% mash = [50, 50];
28% [matrix, m1, m2] = get_fuzzy_matrix('mtr_fuzzy_dsn.fis', 2, range, mash);
29%
