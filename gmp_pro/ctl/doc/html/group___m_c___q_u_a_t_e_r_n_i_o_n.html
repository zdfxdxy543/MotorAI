GMP_STATIC_INLINE ctl_quaternion_t ctl_qmul ( ctl_quaternion_t a,
ctl_quaternion_t b )

Multiplies two quaternions (Hamilton product).

The multiplication is performed as:

\[q_a q_b = (w_a w_b - x_a x_b - y_a y_b - z_a z_b) + \\ (w_a x_b + x_a w_b + y_a z_b - z_a y_b)i + \\ (w_a y_b - x_a z_b + y_a w_b + z_a x_b)j + \\ (w_a z_b + x_a y_b - y_a x_b + z_a w_b)k \]

Parameters
aThe first quaternion.
bThe second quaternion.
Returns
The result of the multiplication (a * b).
