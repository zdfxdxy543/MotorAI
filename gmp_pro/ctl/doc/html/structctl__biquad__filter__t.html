Detailed Description

A second-order infinite impulse response (IIR) general filter.

This module implements a standard biquad filter using the Direct Form I structure. The specific filter type (e.g., low-pass, high-pass) is determined by the 'a' and 'b' coefficients, which are calculated by the various initialization functions provided below.

The Z-domain transfer function is:

\[H(z) = \frac{b_0 + b_1z^{-1} + b_2z^{-2}}{1 + a_1z^{-1} + a_2z^{-2}} \]

The corresponding difference equation is:

\[y(n) = b_0x(n) + b_1x(n-1) + b_2x(n-2) - a_1y(n-1) - a_2y(n-2) \]
