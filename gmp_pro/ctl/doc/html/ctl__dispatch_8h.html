Detailed Description

Provides the core task dispatcher for the CTL motor control framework.

Author
Javnson (javns.nosp@m.on@z.nosp@m.ju.ed.nosp@m.u.cn)

This file contains the main entry point function, typically called from a periodic timer interrupt (ISR). It orchestrates the execution of the different stages of the control loop: sensor input, core algorithm execution, and actuator output. The dispatcher is highly configurable via preprocessor macros to support different scheduling strategies.
