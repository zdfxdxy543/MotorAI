Detailed Description

Header file for a 4-channel data logging module (floating-point).

This module provides functionality to log data from four different sources into separate buffers based on a trigger condition. The trigger is a rising edge on the first input channel crossing a specified threshold. The data logging can be down-sampled using a prescaler.
