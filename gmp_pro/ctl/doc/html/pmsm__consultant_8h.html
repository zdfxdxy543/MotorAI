Detailed Description

Defines data structures and utility functions for PMSM motor parameters.

Author
Javnson (javns.nosp@m.on@z.nosp@m.ju.ed.nosp@m.u.cn)

This file provides "consultant" structures for key nameplate and design parameters of a PMSM, along with functions to calculate derived constants, estimate controller tuning parameters, and convert parameters to the per-unit system.
