Detailed Description

Data structure for a Quasi-Resonant (QR) controller.

A non-ideal resonant controller with finite gain at the resonant frequency, which improves stability and robustness to frequency variations.

Continuous-time transfer function:

\[ G(s) = k_r \frac{2\omega_c s}{s^2 + 2\omega_c s + \omega_r^2} \]
