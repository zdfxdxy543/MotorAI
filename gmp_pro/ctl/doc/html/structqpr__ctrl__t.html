Detailed Description

Data structure for a Quasi-Proportional-Resonant (QPR) controller.

Combines a proportional gain with a quasi-resonant controller.

Continuous-time transfer function:

\[ G(s) = K_p + K_r \frac{2\omega_c s}{s^2 + 2\omega_c s + \omega_r^2} \]
