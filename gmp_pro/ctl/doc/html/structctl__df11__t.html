Detailed Description

Data structure for the DF11 controller.

This structure implements a first-order IIR filter using the Direct Form I topology.

The transfer function is:

\[ H(z) = \frac{b_0 + b_1 z^{-1}}{1 + a_1 z^{-1}} \]
