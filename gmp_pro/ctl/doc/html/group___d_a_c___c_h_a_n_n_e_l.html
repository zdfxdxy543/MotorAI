Detailed Description

A module for handling a single DAC output channel.

This module takes a control value, applies gain and bias, and scales it to the appropriate raw integer format for a DAC peripheral. It supports both fixed-point and floating-point data types. This module converts a logical control value into a raw digital value suitable for being written to a DAC hardware register. The conversion follows the formula:

\[\text{value}_{\text{final}} = (\text{value}_{\text{raw}} \cdot \text{gain}) + \text{bias} \]
