Detailed Description

Defines the state machine and core object for the CTL-Nano framework.

Author
Javnson (javns.nosp@m.on@z.nosp@m.ju.ed.nosp@m.u.cn)

This file provides the top-level structure for managing the lifecycle of the motor controller. It includes a state machine to handle states like pending, calibration, ready, online, and fault. It also defines the main control object that holds all system-level state and parameters.
