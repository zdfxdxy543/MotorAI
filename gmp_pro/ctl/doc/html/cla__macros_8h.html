Detailed Description

Defines a set of macros for floating-point arithmetic using the TI CLA Math library.

This file provides the concrete implementation of the control library's math functions using the Texas Instruments C2000 Control Law Accelerator (CLA) math library. It maps the generic control math macros to their corresponding hardware-accelerated CLA functions and intrinsics.
