GMP_STATIC_INLINE void ctl_matrix2_eigenvalues_complex ( ctl_matrix2_t mat,
ctl_complex_t * eigenvalue1,
ctl_complex_t * eigenvalue2 )

Calculates the eigenvalues of a 2x2 matrix.

It solves the characteristic equation: \( \lambda^2 - \text{trace}(A)\lambda + \text{det}(A) = 0 \). The eigenvalues can be real or a complex conjugate pair.

Parameters
[in]matThe input matrix.
[out]eigenvalue1Pointer to the first complex number to store an eigenvalue.
[out]eigenvalue2Pointer to the second complex number to store an eigenvalue.
