Detailed Description

A collection of common discrete filters for signal processing.

This file contains implementations for a standard second-order IIR filter (Biquad) and a set of helper functions to initialize it as various common filter types (Low-pass, High-pass, Band-pass, Notch, etc.). The coefficient calculations are based on the Audio EQ Cookbook formulas, derived from the Bilinear Transform with frequency pre-warping.

Calculates the linear gain (magnitude) of the biquad filter at a specific frequency.

This function evaluates the filter's magnitude response |H(e^(j¦ŘT))| at the given frequency f. To convert the result to decibels (dB), use the formula: Gain_dB = 20 * log10(linear_gain). Key Steps:

  1. Calculate the normalized angular frequency: ¦ŘT = 2*pi*f / fs.
  2. Evaluate the complex numerator N(¦Ř) and denominator D(¦Ř).
  3. Calculate the magnitude of the numerator: |N(¦Ř)| = sqrt(real(N)^2 + imag(N)^2).
  4. Calculate the magnitude of the denominator: |D(¦Ř)| = sqrt(real(D)^2 + imag(D)^2).
  5. The total gain is |N(¦Ř)| / |D(¦Ř)|.
    Parameters
    [in]objPointer to the biquad filter instance.
    [in]fsSampling frequency (Hz).
    [in]fThe frequency at which to calculate the gain (Hz).
    Returns
    parameter_gt The linear gain (magnitude).

Calculates the phase lag of the biquad filter at a specific frequency.

This function evaluates the filter's complex frequency response H(e^(j¦ŘT)) at the given frequency f. Key Steps:

  1. Calculate the normalized angular frequency: ¦ŘT = 2*pi*f / fs.
  2. Evaluate the complex numerator N(¦Ř) = b0 + b1*e^(-j¦ŘT) + b2*e^(-j2¦ŘT).
  3. Evaluate the complex denominator D(¦Ř) = 1 + a1*e^(-j¦ŘT) + a2*e^(-j2¦ŘT).
  4. Calculate the phase of the numerator and denominator using atan2.
  5. The total phase is phase(N) - phase(D).
  6. The phase lag is the negative of the total phase.
    Parameters
    [in]objPointer to the biquad filter instance.
    [in]fsSampling frequency (Hz).
    [in]fThe frequency at which to calculate the phase lag (Hz).
    Returns
    parameter_gt The phase lag in radians. A positive value indicates lag.

Initializes the biquad filter as a High-Shelf Filter.


The High Self filter is the opposite of the Low Self filter, as it is used to enhance or attenuate all frequencies above a specified "inflection point frequency". S-Domain Transfer Function:

\[ H(s) = A \cdot \frac{s^2 + \frac{\sqrt{A}}{Q}\omega_0 s + \omega_0^2}{s^2 + \frac{1}{\sqrt{A}Q}\omega_0 s + \omega_0^2} \]

where

\[ A = 10^{gain_db} \]

Initializes the biquad filter as a Low-Pass Filter (LPF).

Characteristics: Passes frequencies below the cutoff frequency fc and attenuates frequencies above it. S-Domain Transfer Function:

\[ H(s) = \frac{\omega_0^2}{s^2 + \frac{\omega_0}{Q}s + \omega_0^2} \]

Implementation Steps:

  1. Calculate the normalized angular frequency omega.
  2. Calculate intermediate variables alpha and cos_w0.
  3. Calculate the denominator coefficient a0_inv for normalization.
  4. Calculate the final a and b coefficients for the difference equation.
    Parameters
    [out]objPointer to the biquad filter instance.
    [in]fsSampling frequency (Hz).
    [in]fcCutoff frequency (Hz).
    [in]QQuality factor. Controls the resonance peak at the cutoff frequency. A value of 0.707 gives a Butterworth response.

Initializes the biquad filter as a Peaking EQ Filter.


Peaking EQ is the most common type of equalization (EQ) used to enhance or attenuate a frequency band centered around a certain central frequency, with a frequency response curve that is bell shaped or peak shaped. S-Domain Transfer Function:

\[ H(s) = \frac{s^2 + \frac{A}{Q}s + \omega_0^2}{s^2 + \frac{1}{AQ}s + \omega_0^2} \]

where

\[ A = 10^{gain_db} \]

[in]fcCenter frequency (Hz).
[in]QQuality factor.
