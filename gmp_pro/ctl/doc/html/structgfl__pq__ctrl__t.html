Detailed Description

P-Q Grid-Following Power Controller.

  • This controller sits on top of the Current Controller. It regulates Active Power (P) and Reactive Power (Q) by adjusting the d-axis and q-axis current references.
  • Topology: P_ref —(-)--> [PID_P] ----> Id_ref Q_ref —(-)--> [PID_Q] ----> Iq_ref
