#define ADC_CURRENT_OFFSET_V   ((1.65))

The ADC voltage level corresponding to zero current.

This value represents the offset in the current sensing circuit. For a bipolar current sensor, this is typically half of the ADC reference voltage. Unit: Volts (V)
