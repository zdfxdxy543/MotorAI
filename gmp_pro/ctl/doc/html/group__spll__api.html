Detailed Description

A Phase-Locked Loop for single-phase grid synchronization.

This file provides an implementation of a SOGI-based PLL, which is a robust method for tracking the phase and frequency of a single-phase AC signal. It is commonly used in grid-tied applications like PFC rectifiers and inverters.

The PLL operates as follows:

  1. A Second-Order Generalized Integrator (SOGI) takes the single-phase input and generates a pair of orthogonal \((\alpha -\beta)\) signals.
  2. A Park transform converts these \((\alpha -\beta)\) signals into a rotating d-q frame using the PLL's estimated phase angle.
  3. A PI controller acts as a loop filter, driving the q-component of the transformed signal to zero.
  4. A Voltage-Controlled Oscillator (VCO) integrates the output of the loop filter to generate the estimated frequency and phase, which closes the loop.
