Initializes the SRF-PLL by automatically calculating the PI parameters based on the desired bandwidth and damping ratio.

Parameters
[out]pllPointer to the PLL structure to be initialized.
[in]f_baseThe nominal grid frequency in Hz (e.g., 50.0 or 60.0).
[in]f_ctrlThe control loop execution frequency (sampling frequency) in Hz (e.g., 10000.0).
[in]voltage_magThe magnitude of the input voltage vector (sqrt(alpha^2 + beta^2)). Set this to 1.0 if the input voltages are already normalized (per-unit).
[in]bandwidth_hzThe desired bandwidth of the PLL in Hz. Recommended range: 10.0 ~ 30.0 Hz.
[in]damping_factorThe damping ratio (zeta). Recommended value: 0.707.

Initializes the SRF-PLL by automatically calculating the PI parameters based on the desired bandwidth and damping ratio.

Parameters

[in]f_base电网基准频率 (e.g., 50.0)
[in]f_ctrl控制循环频率/采样频率 (e.g., 10000.0)
[in]voltage_mag输入的电压矢量模长 (sqrt(alpha^2 + beta^2))。如果输入已经标幺化，则填 1.0。
[in]bandwidth_hz期望的 PLL 带宽 (Hz)。推荐值: 10.0 ~ 30.0 Hz
[in]damping_factor阻尼比。推荐值: 0.707

Executes one step of the DSOGI-PLL algorithm.

  1. Filters input voltages using SOGI (Band-Pass & Quadrature generation).
  2. Calculates Positive Sequence Components (PNSC).
  3. Feeds the clean positive sequence into the SRF-PLL.

[in]alphaRaw measured alpha-axis voltage.
[in]betaRaw measured beta-axis voltage.
Returns

GMP_STATIC_INLINE ctrl_gt ctl_step_pll_3ph ( srf_pll_t * pll,
ctrl_gt alpha,
ctrl_gt beta )

Executes one step of the three-phase PLL algorithm.

This function performs a Park transform on the input alpha-beta voltages using the estimated angle, calculates the q-axis error, updates the PI controller, and integrates the frequency to get the new angle.

Parameters
[out]pllPointer to the srf_pll_t structure.
[in]alphaThe measured alpha-axis voltage.
[in]betaThe measured beta-axis voltage.
Returns
The newly estimated grid angle (theta) in per-unit format.
