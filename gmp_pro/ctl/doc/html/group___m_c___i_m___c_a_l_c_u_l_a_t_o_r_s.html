parameter_gt i_sq,
parameter_gt rotor_flux_mag )

Estimates the required slip speed for a given torque-producing current (Isq).

This is used in indirect field-oriented control (IFOC) to calculate the slip. Formula: \(\omega_{sl} = \frac{R_r}{L_r} \frac{L_m i_{sq}}{\lambda_{dr}}\) Note: This simplified version assumes rotor flux is aligned with d-axis, so \(\lambda_r = \lambda_{dr}\)

Parameters
[in]im_dsnPointer to the populated motor consultant structure.
[in]i_sqThe target q-axis (torque-producing) current in Amperes (A).
[in]rotor_flux_magThe target rotor flux magnitude in Weber (Wb).
Returns
The estimated slip speed in electrical rad/s.

Estimates PI controller parameters for the current loop.

Uses the bandwidth method for tuning. The plant is modeled as a first-order system with transient inductance \(\sigma L_s\) and resistance \(R_s\). Formulas:

\[ K_p = \alpha_c \sigma L_s \]

\[ K_i = \alpha_c R_s \]

Parameters
[in]im_dsnPointer to the populated motor consultant structure.
[in]bandwidth_hzThe desired closed-loop bandwidth in Hertz (Hz).
[out]kpPointer to store the calculated proportional gain.
[out]kiPointer to store the calculated integral gain.

GMP_STATIC_INLINE void ctl_acm_tune_speed_pi_params ( const im_dsn_consultant_t * im_dsn,

Estimates PI controller parameters for the speed loop.

Uses the bandwidth method. Assumes the current loop is significantly faster. The torque constant Kt is calculated internally. Formulas:

\[ K_t = 1.5 \cdot p \cdot \frac{L_m}{L_r} \lambda_r \]

\[ K_p = \frac{\alpha_s J}{K_t} \]

\[ K_i = \frac{\alpha_s B}{K_t} \]

Parameters
[in]im_dsnPointer to the populated motor consultant structure.
[in]bandwidth_hzThe desired closed-loop bandwidth in Hertz (Hz).
[in]rotor_flux_magThe target rotor flux magnitude in Weber (Wb) for calculating Kt.
[out]kpPointer to store the calculated proportional gain.
[out]kiPointer to store the calculated integral gain.

GMP_STATIC_INLINE parameter_gt ctl_consult_acm_estimate_slip ( im_dsn_consultant_t * acm,
parameter_gt V_phase_rms,
parameter_gt freq_Hz,
parameter_gt T_load_Nm )

Estimates the motor slip based on its design parameters and a given load torque.

This function uses the Kloss equation to provide a good approximation of the slip for a given steady-state load torque. It calculates the breakdown torque and the corresponding

critical slip first, then solves for the operational slip. Kloss equation:

\[ \frac{T_e}{T_{max}} = \frac{2}{s/s_k + s_k/s} \]

Parameters
acmPointer to the induction motor design parameters structure.
V_phase_rmsThe operating RMS phase voltage [V].
freq_HzThe operating synchronous frequency [Hz].
T_load_NmThe load torque applied to the motor shaft [N·m].
Returns
The estimated slip (a value between 0 and 1). Returns a negative value if the load torque exceeds the motor's breakdown torque.
