Detailed Description

Data structure for holding PMSM design (model) parameters.

These parameters are used in the mathematical model of the motor for FOC.

Pole pairs: \(p\), Stator Resistance: \(R_s\), d-axis Inductance: \(L_d\), q-axis Inductance: \(L_q\), PM Flux: \(\psi_f\), Inertia: \(J\), Damping: \(B\).
