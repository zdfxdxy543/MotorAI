Detailed Description

Data structure for the DF13 controller.

This structure implements a third-order IIR filter using the Direct Form I topology.

The transfer function is:

\[ H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2} + b_3 z^{-3}}{1 + a_1 z^{-1} + a_2 z^{-2} + a_3 z^{-3}} \]
