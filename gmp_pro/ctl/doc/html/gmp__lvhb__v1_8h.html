This header file contains calibration constants for the voltage and current sensors on the Diansai v1 hardware platform, dated 2025-07-20. These macros are used to convert raw ADC values into physical units (Volts and Amperes).

#define DSV1_CURRENT_SENSOR_BIAS   ((float)(0.9))

The bias voltage of the current sensor op-amp circuit, in Volts.

This is the output voltage of the sensor circuit when the input current is zero. For bidirectional sensors, this is typically half of the ADC reference voltage.

◆ DSV1_CURRENT_SENSOR_GAIN

#define DSV1_CURRENT_SENSOR_GAIN   ((float)(0.005 * 20.0))

The gain of the current sensor circuit.

This value is typically the product of the shunt resistor value and the amplifier gain (e.g., R_shunt * G_amp). The formula to convert ADC reading to current is: I_real = (ADC_V - I_bias) / I_gain.

◆ DSV1_VOLTAGE_SENSOR_GAIN

#define DSV1_VOLTAGE_SENSOR_GAIN   ((float)(2.2 / 202.2))

The gain of the voltage sensor circuit.

This value typically represents the voltage divider ratio (e.g., R2 / (R1 + R2)). The formula to convert ADC reading to voltage is: V_real = (ADC_V - V_bias) / V_gain.
