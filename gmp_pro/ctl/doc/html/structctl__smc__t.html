Detailed Description

Data structure for the Sliding Mode Controller.

The controller takes two state variables as input: x1 (the controlled variable) and x2 (its derivative).

The sliding surface is defined as:

\[s = \lambda x_1 + x_2 \]

The control law is a form of reaching law:

\[u = u_{eq} + u_{sw} = (\eta_1 x_1 + \eta_2 x_2) + \rho \cdot \text{sgn}(s) \]

where the gains \( \eta_1 \) and \( \eta_2 \) are switched based on the signs of the sliding surface and the state variables.
