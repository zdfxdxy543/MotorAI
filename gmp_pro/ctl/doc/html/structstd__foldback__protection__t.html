Detailed Description

Data structure for a foldback current protection module.

This module implements a foldback overcurrent protection scheme. When the output voltage drops due to an overload, the current limit is proportionally reduced, minimizing power dissipation during fault conditions like a short circuit. The output of this module is a dynamic current limit reference, intended to be fed into the current control loop.
