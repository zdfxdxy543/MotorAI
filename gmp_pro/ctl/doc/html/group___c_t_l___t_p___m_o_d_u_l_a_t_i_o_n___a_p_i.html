Determines the current direction zone for dead-time compensation.

  • Parameters
    i_sampleCurrent sample value (Per-Unit or Amps).

    hysteresisThe hysteresis band width (Zone 4 width).
    last_dirPointer to the memory of the last state (-1, 0, 1).
