Detailed Description

Decoupling voltage calculation for Permanent Magnet Synchronous Motors.

This module contains functions to calculate the cross-coupling voltage terms in the d-q reference frame for both PMSM and Induction Motors. These feed-forward terms are essential for achieving high-performance current control. Functions are provided for both SI unit and per-unit system calculations.

This function computes the cross-coupling terms that should be added to the PI controller outputs to decouple the d and q axis dynamics. Formulas:

\[ V_{d,ff} = - \omega_e \cdot L_q \cdot i_q \]

\[ V_{q,ff} = \omega_e \cdot (L_d \cdot i_d + \psi_f) \]
