Detailed Description

A self-tuning filter that minimizes the mean square error.

This file implements an LMS adaptive filter. The filter continuously adjusts its internal coefficients (weights) to minimize the mean square error between its output and a desired reference signal. It is widely used for applications like active noise cancellation, system identification, and echo cancellation. The core of the algorithm is the update rule:

\[ W(n+1) = W(n) + mu * e(n) * X(n) \]

reference: https://zhuanlan.zhihu.com/p/358236441

Initializes the LMS adaptive filter.

This function allocates memory for the filter's weights and data buffer. The user is responsible for calling ctl_destroy_lms_filter to free this memory.

Parameters
[out]lmsPointer to the LMS filter instance.
[in]orderThe number of filter taps (coefficients).
[in]muThe step-size (learning rate). Must be chosen carefully to ensure stability.
