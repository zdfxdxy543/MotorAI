GMP_STATIC_INLINE void ctl_step_spd_calc ( spd_calculator_t * sc)

Executes one step of the speed calculation.

This function should be called periodically. It calculates the speed based on the change in position since the last call, filters it, and updates the output. The internal divider determines the actual execution rate.

Parameters
[in,out]scPointer to the speed calculator structure.
