Detailed Description

Implements a SOGI with an additional loop to reject DC offset.

This module extends the standard SOGI by adding a DC estimation loop. Ideally, the alpha channel of a standard SOGI rejects DC (gain=0 at 0Hz). However, any DC offset in the input passes through to the beta channel (integrator), causing low-frequency oscillation in the DQ frame (50/60Hz ripple).

This module estimates the DC component and subtracts it from the input before feeding it to the SOGI core, ensuring pure AC signals for alpha and beta outputs.

generally,

\[ k_{dc} \approx 0.5 ~ 1.0 \]

Transfer Functions (Simplified structural view):

Direct Output (Alpha) D(s):

\[ \frac{D(s)}{R(s)} = \frac{k \omega_0 s^2}{s^3 + (k + k_{dc})\omega_0 s^2 + \omega_0^2 s + k_{dc}\omega_0^3} \]

Quadrature Output (Beta) Q(s):

\[ \frac{Q(s)}{R(s)} = \frac{k \omega_0^2 s}{s^3 + (k + k_{dc})\omega_0 s^2 + \omega_0^2 s + k_{dc}\omega_0^3} \]

DC Estimation Output DC(s):

\[ \frac{DC(s)}{R(s)} = \frac{k_{dc} \omega_0 (s^2 + \omega_0^2)}{s^3 + (k + k_{dc})\omega_0 s^2 + \omega_0^2 s + k_{dc}\omega_0^3} \]

where \( k_{dc} \) is the DC rejection gain.
