GMP_STATIC_INLINE void ctl_ct_clarke ( const ctl_vector3_t * abc,
GMP_CTL_OUTPUT_TAG ctl_vector3_t * ab )

Performs the Clarke transformation from a 3-phase (ABC) to a 2-phase stationary (alpha-beta) reference frame.

\[ i_\alpha = \frac{2}{3} i_a - \frac{1}{3} i_b - \frac{1}{3} i_c \]

\[ i_\beta = \frac{1}{\sqrt{3}} i_b - \frac{1}{\sqrt{3}} i_c \]

\[ i_0 = \frac{1}{3} (i_a + i_b + i_c) \]

Parameters
[in]abcPointer to the input 3-phase vector.
[out]abPointer to the output alpha-beta-0 vector.
