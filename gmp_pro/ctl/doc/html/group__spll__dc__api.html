Detailed Description

A Phase-Locked Loop with DC Offset Rejection for single-phase grid synchronization.

This implementation uses a SOGI-DC block to generate orthogonal signals. The DC rejection loop inside the SOGI-DC block estimates and subtracts the input DC offset, preventing low-frequency oscillations (50/60Hz) in the d-q frame outputs.
