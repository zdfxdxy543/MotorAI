Detailed Description

A module to generate a constant frequency signal output.

This file includes modules for generating constant frequency, sloped frequency, and a full constant V/F profile. These are typically used for simple open-loop control of AC motors. This module uses a ramp generator to produce a continuously incrementing angle (position) at a fixed frequency.
