Detailed Description

Defines the parameters for a specific AC Induction Motor (ACM).

This file contains the electrical, mechanical, and operational parameters for a 24V, 4-pole AC induction motor. These macros are intended to be used throughout the motor control application to configure various algorithms and safety limits.
