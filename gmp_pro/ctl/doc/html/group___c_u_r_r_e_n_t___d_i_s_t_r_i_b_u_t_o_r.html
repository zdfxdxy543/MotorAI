Detailed Description

Distributes a total current command into Id and Iq components via a LUT.

This module takes a total current magnitude command (Im) and distributes it into d-axis (Id) and q-axis (Iq) components. The distribution is determined by a current angle (alpha), which can either be a fixed constant or interpolated from a 1D look-up table using the functions provided by the surf_search module. This is typically used to implement strategies like MTPA (Maximum Torque Per Ampere) where the optimal current angle varies with the total current.
