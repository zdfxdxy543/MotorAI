Detailed Description

Defines a consultant module to calculate base values for a motor's per-unit system.

Author
Javnson (javns.nosp@m.on@z.nosp@m.ju.ed.nosp@m.u.cn)

The per-unit system simplifies analysis and control of power systems by normalizing all quantities with respect to a common set of base values. This module takes fundamental motor ratings (power, voltage, frequency) and computes a consistent set of base values for current, impedance, flux, torque, etc.
